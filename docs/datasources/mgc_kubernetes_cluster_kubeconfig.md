# mgc_kubernetes_cluster_kubeconfig (Datasource)

## Introduction

The mgc_kubernetes_cluster_kubeconfig data source allows you to retrieve the kubeconfig file for a Kubernetes cluster managed by Magalu Cloud. This kubeconfig file is essential for authenticating and interacting with your Kubernetes cluster using kubectl and other Kubernetes clients. For more information see the [official documentation](https://docs.magalu.cloud/docs/kubernetes/overview).

## Example Usage

```hcl
provider "mgc" {
  region = "br-se1"
}

data "mgc_kubernetes_cluster_kubeconfig" "cluster" {
  cluster_id = "your-cluster-id"
}

resource "local_file" "kubeconfig" {
  content  = data.mgc_kubernetes_cluster_kubeconfig.cluster.kubeconfig
  filename = "${path.module}/kubeconfig"
}
```

-> For more examples, you can visit the [Magalu Cloud Terraform example repository.](https://github.com/MagaluCloud/terraform-examples)

## Use Cases
1. **Cluster Management:** Easily manage your Kubernetes cluster by fetching the kubeconfig file and using it with kubectl or other Kubernetes tools.
2. **CI/CD Pipelines:** Integrate the kubeconfig retrieval into your CI/CD pipelines to automate deployments and other cluster operations.
